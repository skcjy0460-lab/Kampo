# 자동차사고 한방치료 안내 Streamlit 앱

환자 정보와 날짜를 입력하면 내원, 추나요법, 약침술, ICT, 자락관법의 적용 구간을
A4 한 페이지 안내서로 출력하는 Streamlit 앱입니다. 첨부 서식의 계산 취지는
참고하되, 화면 및 출력 디자인은 새로 구성했습니다.

출력 안내서 하단에는 입원, 치료 항목, 첩약, 초기 통증 변화 및 보험 서류에 관한
교통사고 치료 Q&A 안내를 함께 표시합니다.

## 실행

```powershell
python -m pip install -r requirements.txt
streamlit run streamlit_app.py
```

왼쪽 입력 영역에서 성명, 생년월일, 사고일, 초진일 및 안내 기준일을 입력한 다음
`A4 인쇄 / PDF 저장` 버튼을 누릅니다. 브라우저 인쇄 설정은 `A4`, `세로`,
`배율 100%`, `여백 없음`을 권장합니다.

## 계산 기준

| 치료 항목 | 기준 날짜 | 표시 내용 |
| --- | --- | --- |
| 내원 / 추나요법 | 사고일 | 1-21일 매일, 22-84일 주 3회, 85-183일 주 2회, 이후 주 1회 / 추나요법 최대 20회 안내 |
| 약침술 | 사고일 | 1-7일 매일, 8-21일 주 3회, 22-70일 주 2회, 이후 주 1회 |
| ICT | 사고일 | 1-17일 외래 1일 1회 2부위·입원 1일 2회 2부위, 18일 이후 외래 1일 1회 1부위·입원 1일 2회 1부위 |
| 자락관법 | 초진일 | 1-7일 매일, 8-21일 주 4회, 이후 주 2회 |

이 앱은 입력 날짜에 따른 **가능 빈도 안내**를 출력합니다. 실제로 이미 시행된
횟수를 알기 위해서는 진료기록 또는 청구 자료 연동이 별도로 필요합니다.

## 기준 관리와 유의사항

기간 및 빈도는 [treatment_rules.py](./treatment_rules.py)에 모아 두었으므로 기준이
변경되면 해당 파일의 규칙 상수만 검토하여 수정할 수 있습니다.

- ICT 및 추나요법 관련 확인 자료: 건강보험심사평가원,
  [자동차보험진료수가에 관한 기준, 심사지침 (2023년 12월판)](https://www.hira.or.kr/ebooksc/2024/01/BZ202401319575775.pdf)
- 내원·추나요법의 기간별 빈도, 약침술 구간 및 자락관법 구간은 제공받은 서식과
  요청 조건을 앱 설정값으로 반영했습니다. 서비스 공개 전 최신 고시 및 심사기준과
  대조 확인해야 합니다.
